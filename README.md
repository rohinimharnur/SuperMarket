Shri Balaji Supermarket (Node + Express + MongoDB)

Quick start

1. Install deps:

   npm install

2. Set environment variables in a `.env` file:

   MONGO_URL=<your mongodb connection string>
   PORT=3001
   SESSION_SECRET=<secret>

3. Run locally:

   node server.js

Hosting guide: see DEPLOY.md for step-by-step instructions to host on Render, Railway or Heroku.
